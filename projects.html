Audio Forensics Framework

PyTorch WavLM Torchaudio Librosa

Capstone project in audio forensics covering speaker profiling, deepfake detection, and environmental sound classification. Used WavLM for speaker attributes (age, gender, accent, emotion) and deepfake detection, along with a CRNN model for environmental sound classification. Models were trained and evaluated on Common Voice, RAVDESS, ASVspoof, and ESC-50.

Talk To Philosophers

RAG LLM FAISS Face Streamlit

Talk To Philosophers is a local RAG-based project that provides a simple Streamlit interface to chat with philosophers like Marcus Aurelius and René Descartes, where user queries are answered by retrieving relevant passages from source texts using FAISS and then building prompts that reflect each philosopher’s tone, worldview, and historical context before generating responses locally; the system is structured as a modular pipeline covering ingestion, indexing, retrieval, and generation, with configurable personas that make it easy to extend while keeping responses grounded in the original texts and consistent with each thinker’s perspective.

Football Player Similarity Search

Machine Learning Python PCA KNN

Finds similar football players from 2024-2025 stats using PCA and KNN. Player data from the top 5 leagues is cleaned and reduced, then compared using cosine similarity to return the 3 most similar players, with goalkeepers handled separately from outfield players for more relevant comparisons.
